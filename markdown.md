# Markdown

## 标题
- \# 一级标题
- \## 二级标题
- \### 三级标题
- \#### 四级标题
- \##### 五级标题
- \###### 六级标题

## 字体

- 斜体
  - `*文本内容*`
  - `_文本内容_`
- 粗体
  - `**文本内容**`
  - `__文本内容__`
- 粗斜体
  - `***文本内容***`
  - `___文本内容___`

## 分隔线

`------` 3个字符或以上即可

## 删除线

`~~文本内容~~`

## 下划线

`<u>文本内容</u>`

## 脚注

- `[^文本内容]` 对文本的补充说明 [^就像这样]
- [^就像这样]:www.google.com ，一般在文章末尾

## 区块

```
> 文本内容
> 内容文本
> 内容摘要
```

> 文本内容
> 内容文本
> 内容摘要
