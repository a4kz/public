# Markdown

## 标题
- \# 一级标题
- \## 二级标题
- \### 三级标题
- \#### 四级标题
- \##### 五级标题
- \###### 六级标题

## 字体

- 斜体
  - `*文本内容*`
  - `_文本内容_`
- 粗体
  - `**文本内容**`
  - `__文本内容__`
- 粗斜体
  - `***文本内容***`
  - `___文本内容___`

## 分隔线

`------` 3个字符或以上即可

## 删除线

`~~文本内容~~`

## 下划线

`<u>文本内容</u>`

## 脚注

- `[^文本内容]` 对文本的补充说明 [^就像这样]
- [^就像这样]:www.google.com ，一般在文章末尾

## 区块

```
> 文本内容
> 内容文本
> 内容摘要
```

> 文本内容
> 内容文本
> 内容摘要

### 可以嵌套列表

> title-1
> - list - 1
> - list - 2
> - list - 3

## 链接

- `[名称](URL)`
  - 这个[Google](https://www.google.com)网站
  
- `<URL>`
  
  - <https://www.google.com>
  
- 事后补全链接

  - This is [Google][1]

  - [1]:https://www.google.com

## 图片

- `![alt](URL)`

## 数学公式

- `$f(x) = sin(x)$`
- `$$f(x) = sin(x)$$`
